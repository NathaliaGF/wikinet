Network+ N10-009 — Domínios e Cobertura

O exame N10-009 tem 5 domínios. Veja o peso de cada um e como o RedesWiki cobre o conteúdo:

1 Networking Concepts 23%

OSI model, TCP/IP, portas e protocolos, tipos de rede (LAN/WAN/MAN/PAN), cloud, virtualização e conceitos de datacenter.

Módulos cobertos: Modelos de Camadas Protocolos Portas Fundamentos
85%
2 Network Implementation 20%

Roteamento, switching (VLANs, STP), wireless (802.11), IPv4/IPv6, sub-redes, NAT/PAT e tipos de cabos/conectores.

Módulos cobertos: Endereçamento IPv6 Equipamentos
70%
3 Network Operations 19%

Monitoramento, documentação, baselines, gerenciamento de configuração, políticas de backup, gerenciamento de licenças.

Módulos cobertos: Troubleshooting
40%
4 Network Security 19%

Ataques (MITM, DDoS, phishing), firewalls, IDS/IPS, VPN, AAA, hardening de dispositivos, segurança wireless.

Módulos cobertos: Segurança Básica Lab MITM
75%
5 Network Troubleshooting 19%

Metodologia de troubleshooting, ferramentas (ping, traceroute, nslookup, netstat, Wireshark), problemas de cabo, wireless e switching.

Módulos cobertos: Troubleshooting Laboratório
80%
Dica N10-009: O exame inclui questões de desempenho (PBQs — Performance-Based Questions) onde você precisa configurar dispositivos virtuais ou resolver problemas num simulador. Pratique com o Laboratório Interativo do RedesWiki e com os laboratórios gratuitos do Packet Tracer.

CCNA 200-301 — Domínios e Cobertura

O CCNA é mais técnico e focado em implementação Cisco. Tem 6 domínios:

1 Network Fundamentals 20%

Interfaces e cabos, OSI e TCP/IP, IPv4 e IPv6, sub-redes, wireless (2.4 GHz vs 5 GHz), virtualização e cloud.

Módulos cobertos: Fundamentos Modelos Endereçamento IPv6
80%
2 Network Access 20%

VLANs, trunking (802.1Q), STP/RSTP, EtherChannel, WLAN (WPA2/3, 802.11), configuração de access points.

Módulos cobertos: Equipamentos
30%
3 IP Connectivity 25%

Roteamento estático, OSPFv2, IPv6 routing, primeiro salto (FHRP), troubleshooting de rotas.

Módulos cobertos: Endereçamento Acesso a um Site Laboratório
50%
4 IP Services 10%

NAT, NTP, DHCP, DNS, SNMP, Syslog, QoS (DSCP, queuing), SSH vs Telnet.

Módulos cobertos: Protocolos NAT
60%
5 Security Fundamentals 15%

Threats, ACLs, port security, DHCP snooping, DAI, VPNs (site-to-site, remote access), AAA com RADIUS.

Módulos cobertos: Segurança Lab MITM + ARP
65%
6 Automation and Programmability 10%

SDN, controller-based networking, REST APIs, Cisco DNA Center, Ansible/Puppet conceitos, JSON/YAML.

Módulos cobertos: Fora do escopo atual do RedesWiki
10%
Atenção CCNA: O exame cobra configuração real de IOS Cisco (linha de comando). Use o Cisco Packet Tracer (gratuito, requer cadastro) ou o GNS3 para praticar CLI. O RedesWiki cobre os conceitos; a prática em CLI é indispensável à parte.

Cronograma de Estudo Sugerido

Network+ — 8 semanas

S1
Fundamentos + Modelos — OSI, TCP/IP, cabos, topologias
S2
Endereçamento — IPv4, sub-redes, CIDR, NAT, IPv6
S3
Protocolos + Portas — DNS, DHCP, HTTP, TCP/UDP, portas comuns
S4
Equipamentos + Switching — Switch, roteador, firewall, VLANs, STP
S5
Segurança — Ataques, VPN, firewalls, AAA, hardening
S6
Wireless + Cloud — 802.11, WPA3, virtualização, IaaS/PaaS/SaaS
S7
Troubleshooting — Metodologia, ferramentas, simulados de falha
S8
Revisão + Simulados — Exames práticos, PBQs, pontos fracos

S1–2

S7–8
IPv6 — Formato, tipos, SLAAC, roteamento OSPFv3
S9–10
IP Services — NAT, DHCP, NTP, SNMP, QoS
S11–12
Segurança — ACLs, port security, DAI, DHCP snooping, VPN
S13–14
Wireless + Automation — 802.11, WLC, SDN, REST API
S15–16
Revisão + Simulados — Simulados completos, revisão de CLI, pontos fracos
